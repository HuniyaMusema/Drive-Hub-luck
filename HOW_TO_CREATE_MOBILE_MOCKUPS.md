# 📱 How to Create Mobile Mockups for Drive Hub Luck

## Quick Guide to Generate Professional Mobile Mockups

---

## Method 1: Using Browser DevTools (Free & Fast)

### Step 1: Open Your App
1. Start your frontend: `cd frontend && npm run dev`
2. Open in Chrome/Edge: `http://localhost:5173`

### Step 2: Enable Device Mode
1. Press `F12` to open DevTools
2. Click the device toggle icon (or press `Ctrl+Shift+M`)
3. Select device: iPhone 14 Pro, Galaxy S21, etc.

### Step 3: Take Screenshots
1. Click the three dots in DevTools
2. Select "Capture screenshot" or "Capture full size screenshot"
3. Save the images

### Step 4: Add Device Frame
Go to **mockuphone.com**:
1. Upload your screenshot
2. Select device (iPhone 14 Pro, etc.)
3. Download the mockup with device frame

---

## Method 2: Using Figma (Professional)

### Setup (One-time)
1. Go to **figma.com** and create free account
2. Search for "iPhone mockup" in Community
3. Duplicate a free mockup template

### Create Mockup
1. Import your screenshots
2. Place them in the device screen area
3. Adjust size to fit
4. Export as PNG/JPG

**Recommended Templates:**
- "iPhone 14 Pro Mockup" by Apple
- "Device Mockups" by Facebook Design
- "Mockup Bundle" by Figma Community

## Method 4: Using Photoshop/GIMP

### Requirements
- Photoshop or GIMP (free)
- Device mockup PSD template

### Steps
1. Download free PSD mockup from:
   - **Freepik.com**
   - **Mockup.world**
   - **Graphicburger.com**

2. Open PSD in Photoshop/GIMP
3. Find "Smart Object" layer
4. Double-click to open
5. Paste your screenshot
6. Save and close
7. Export final mockup

## 🎯 Pro Tips

### For Best Results

1. **Use Actual Device Dimensions**
   - iPhone 14 Pro: 393 × 852 px
   - iPhone 14 Pro Max: 430 × 932 px
   - Samsung Galaxy S21: 360 × 800 px

2. **Clean Screenshots**
   - Hide scrollbars
   - Use real data (not Lorem Ipsum)
   - Show realistic content
   - Include status bar

3. **Consistent Styling**
   - Same device for all screens
   - Same background color/gradient
   - Same orientation
   - Same zoom level

4. **Context Matters**
   - Show time as 9:41 (Apple standard)
   - Full battery
   - Good signal strength
   - Realistic notifications

## 🎨 Example Mockup Workflow

### Complete Process (15 minutes)

1. **Prepare** (2 min)
   - Start dev server
   - Open in browser
   - Enable device mode

2. **Capture** (5 min)
   - Take 6-8 key screenshots
   - Different pages/states
   - Save to folder

3. **Frame** (5 min)
   - Upload to shots.so
   - Add iPhone 14 Pro frame
   - Choose gradient background
   - Download all

4. **Compose** (3 min)
   - Open Figma/Canva
   - Arrange mockups
   - Add title/description
   - Export final image

**Result:** Professional mobile mockup ready to share! 🎉
